# ✈️ SkyWings Analytics Dashboard

A modern travel analytics dashboard built using HTML, CSS, and JavaScript.

## 🚀 Features
- Upload Excel dataset
- Dynamic charts (Bar, Line, Pie)
- KPI Metrics (Revenue, Avg Booking)
- Season Filter (Peak / Off-Peak)
- Search & Table View
- CSV Download
